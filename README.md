# Fashion-MNIST MLP Classifier from Scratch

This repository contains a manual implementation of a 3-layer Multilayer Perceptron (MLP) built entirely with NumPy, without using any autograd frameworks like PyTorch or TensorFlow.

## Requirements
`pip install numpy matplotlib seaborn scikit-learn`

## How to Run
1. To train the model and run hyperparameter search:
   `python main.py`
2. The script will automatically download the dataset, train the models, save the best weights to `best_model.pkl`, and plot the learning curves.
